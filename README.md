# 🌆 Urban Heat Island Geospatial Analysis & Visualization

🔗 Live App: https://urban-heat-island-mumbai.streamlit.app/

## 📌 Overview

This project analyzes Urban Heat Island (UHI) patterns in Mumbai using Landsat 9 satellite data.

It enables interactive exploration of vegetation, temperature, and urban heat intensity through a web-based dashboard.
Interactive geospatial dashboard analyzing Urban Heat Island (UHI) patterns using Landsat satellite data with NDVI, temperature, and correlation analysis.

## 🚀 Features

- 🗺️ Interactive map with raster overlays  
- 🎨 Dynamic color scales (NDVI, Temperature, UHI)  
- 📊 Summary statistics (Mean, Min, Max)  
- 🌱 Vegetation coverage analysis  
- 🔥 UHI hotspot detection  
- 🔄 NDVI vs Temperature comparison  
- 📉 Correlation insights  
- ☁️ Deployed on Streamlit Cloud  

## 🛠️ Tech Stack

- Python  
- Streamlit  
- NumPy  
- Matplotlib  
- Folium  
- Geospatial Data (Landsat 9)

## 📚 Key Learnings

- Working with geospatial raster data  
- Calculating NDVI and UHI indices  
- Building interactive dashboards  
- Handling real-world deployment issues  
- Cloud deployment using Streamlit  

## ⭐ Conclusion

This project demonstrates how satellite data can be transformed into actionable urban climate insights through data visualization and analysis.
